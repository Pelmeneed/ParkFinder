# ParkFinder

Мобильное приложение для поиска ближайших парковочных мест в городе Нижний Тагил.

## 📱 Функциональные возможности

- Отображение карты с центром в Нижнем Тагиле
- Автоматическое определение текущего местоположения пользователя
- Отображение парковочных мест в заданном радиусе
- Выбор радиуса поиска (50 м, 100 м, 250 м, 500 м, 750 м, 1 км)
- Интерактивные маркеры с названием и расстоянием до парковки
- Управление картой (масштабирование, поворот, перемещение)

## 🚀 Быстрый старт

### Установка

1. Скачайте APK-файл из раздела [Releases](https://github.com/yourusername/parkfinder/releases)
2. Разрешите установку из неизвестных источников в настройках Android
3. Откройте приложение и предоставьте разрешение на доступ к геолокации

### Первое использование

- При запуске карта центрируется на Нижнем Тагиле
- Приложение автоматически определит ваше местоположение и переместит камеру
- По умолчанию радиус поиска – **250 метров**

## 🎮 Руководство пользователя

### Выбор радиуса поиска

1. Нажмите кнопку **«Выбрать радиус»** в верхнем левом углу
2. Выберите значение из списка:
   - 50 м
   - 100 м
   - 250 м (по умолчанию)
   - 500 м
   - 750 м
   - 1 км
3. Диалог закроется автоматически
4. Нажмите кнопку **«Найти»** внизу экрана

### Взаимодействие с картой

| Действие | Жест |
|----------|------|
| Масштабирование | Сведение / разведение двух пальцев |
| Поворот | Вращение двумя пальцами |
| Перемещение | Перетаскивание одним пальцем |

### Просмотр информации о парковке

Нажмите на любой маркер – отобразится всплывающая подсказка с названием и расстоянием (в метрах).

## 📋 Системные требования

- **Android**: версия 13 и выше
- **Обязательно**: включённый GPS/GLONASS
- **Интернет**: требуется для загрузки карт (данные о парковках хранятся локально)

## 🛠️ Технические детали

- **Язык разработки**: Kotlin
- **Картографический сервис**: Google Maps SDK
- **Геолокация**: Google Play Services Location
- **Хранение данных**: JSON-файл в ресурсах приложения
